# Reports

ITIL practices: Measurement and Reporting, Continual Improvement.

## Navigation Structure

- Route: `/reports`
- Secondary tabs: ITIL KPIs, Operational Reports, Compliance Reports, Scheduled Jobs, Downloads
- Detail tabs: Parameters, Results, Delivery, Audit

## Menus

- Header menu:
  - Run report
  - Create custom report
  - Manage schedules
  - Export report catalog
- Row action menu:
  - Run now
  - Edit parameters
  - Clone report
  - Schedule delivery
  - Archive report
- Bulk action menu:
  - Bulk run
  - Bulk schedule update
  - Bulk archive
  - Bulk export metadata
- Detail action menu:
  - Download output
  - Share report link
  - Subscribe recipients
  - Add compliance tag

## Required UI Elements

- Report catalog with ownership, frequency, data scope, last run status.
- Parameter builder (date range, teams, services, priorities, format).
- Run history with job status and error reason.
- Download center with retention information.
- Scheduled delivery table and recipient list.

## Required Features

- Prebuilt ITIL KPI packs (MTTR, FCR, change success, SLA attainment, backlog age).
- Custom report templates and saved parameter sets.
- Scheduled delivery by email/webhook.
- Audit trail for report generation and downloads.

## KPIs

- Report execution success rate, schedule adherence, report usage rate.
